# Remote auth

The package allow remote authentication using the auth provider of laravel
## Start 🚀

```
composer install leugin/remote-auth
```

### Pre-requisitos 📋

```
laravel:^10.47
```

### Instalacion 📋

```
Add RemoteAuthServiceProvider to config\app.php file
```
